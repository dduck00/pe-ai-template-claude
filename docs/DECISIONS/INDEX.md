## 상태 정의

| 상태 | 설명 |
|------|------|
| `proposed` | 제안됨, 미확정 |
| `accepted` | 확정, 현행 적용 중 |
| `deprecated` | 폐기 (대체 ADR 참조) |
| `superseded` | 후속 ADR로 대체됨 (대체 ADR 번호 기록) |

## 갱신 규칙

- ADR 신규 작성 시 이 인덱스에 행을 추가한다.
- ADR 상태 변경 시 이 인덱스도 동기화한다.
- 현행 결정만 빠르게 파악하려면 `accepted` 상태만 필터링한다.
